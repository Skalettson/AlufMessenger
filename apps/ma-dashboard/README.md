# Aluf MA Dashboard

Панель управления для разработчиков **Aluf Mini-Apps**.

## 🚀 Возможности

- **Управление приложениями** - создание, редактирование, удаление Mini-Apps
- **Аналитика** - просмотры, пользователи, события
- **Монетизация** - платежи, подписки, доходы
- **Настройки** - конфигурация, permissions, webhook
- **Тестирование** - предпросмотр, отладка

## 📦 Установка

```bash
pnpm install
pnpm dev
```

## 🎯 Страницы

### Dashboard
Обзор всех приложений с ключевыми метриками.

### Apps
Список всех Mini-Apps с возможностью создания новых.

### App Details
Детальная информация о приложении:
- Основная информация
- Настройки
- Версии
- Аналитика
- Отзывы

### Analytics
Подробная аналитика:
- DAU/MAU
- Сессии
- События
- Воронки

### Monetization
Управление платежами:
- Тарифы
- Подписки
- Транзакции
- Выплаты

### Settings
Настройки разработчика:
- Профиль
- API ключи
- Webhook
- Команда
